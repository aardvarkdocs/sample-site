# Contributing

Thanks for your interest.

## Issues are welcome

If you find a bug in this example, have a suggestion, or something is unclear,
please [open an issue](../../issues/new). We read them.

## Pull requests are not accepted

This repository is a one-way, automatically published mirror of an upstream
source. Any change committed here would be overwritten by the next sync, so pull
requests cannot be merged and are closed automatically.

To propose a change to the example itself, open an issue describing it — fixes are
made upstream and flow back here on the next publish.

## Starting your own project

Use the **Use this template** button at the top of the repository to create your
own copy. Your repository is independent of this one; edit it however you like.
