# Standardized code-block Copy & Download buttons

Every code block — on the page, in the file-tree modal, and in OpenAPI request/response
samples — now shares one **Copy** and **Download** affordance, so the interaction is
identical everywhere a reader meets code. See the [components](/components/) library.
